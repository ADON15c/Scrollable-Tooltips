# Scrollable-Tooltips
A clientsided minecraft mod allowing long item tooltips to be scrolled through

### Example:
![alttext](ReadmeAssets/ScrollableTooltipsDemo.png)

### Requirments:
- Fabric Modloader
- Minecraft 1.18.2 (untested on previous versions)
